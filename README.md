# Online Make Up Store | Adorn

A simple navigation template for an online makeup store. This project provides a clean HTML structure and styling for an elegant landing page with a responsive navigation bar.

---

## Features

- Clean, modern HTML5 markup
- Responsive navigation menu
- Example logo and favicon usage
- Sample section placeholders
- Designed for easy customization

---

## Project Structure

```
Navigation-Template_HTML/
│
├── main.html
├── main_style.css
└── Images/
   ├── Logo.png
   └── Cute-Fox-icon.png
```

---

## Getting Started

1. **Clone the repository**  
   ```
   git clone https://github.com/Amature-Learner24/Navigation-Template_HTML.git
   ```

2. **Open `main.html` in your browser**  
   Double-click the `main.html` file, or open it in your preferred browser.

3. **Customize**  
   - Edit the HTML and CSS to suit your own branding.
   - Replace logo and favicon images within the `Images/` folder.
